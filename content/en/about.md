    *   AIClient-2-API is a robust API proxy service. It wraps client-only large model capabilities (like Gemini CLI, Antigravity, Qwen Code, Kiro) into a local OpenAI-compatible interface. Plus, it handles account pool management, intelligent polling, automatic failover, and system prompt management like a boss.
*   **[Open Source Contribution/OpenClaw-Docker-CN-IM](https://github.com/justlovemaki/OpenClaw-Docker-CN-IM)**:
    *   OpenClaw-Docker-CN-IM is a Docker image integrating OpenClaw's Chinese IM plugins. It comes pre-installed and configured with plugins for major Chinese IM platforms like Feishu, DingTalk, QQ Robot, and WeChat Work. This makes it super quick to deploy a multi-platform AI bot gateway.
